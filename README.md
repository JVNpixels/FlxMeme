### A HaxeFlixel project where you can make memes!
NOTE: It is only limited to those early 2000's style type of memes.
Version: v1.0.2

### You can:
- Customize Texts.
- Angle Texts.
- Load Images.
- Angle Images.
- Hide UI elements.
- Use helper buttons for resetting to default values.
- Use more helper buttons to center texts/images.
